<h1 align="center">Vue Vite Boilerplate</h1>

## Features

### Core
- Vue 3
- Vue Router 4
- Pinia
- Vite 2
- TypeScript

### Routing
- Layout system

### Style
- SCSS

### Code Style
- Eslint
- Prettier
- Standard

### Commit convention
- Adapted Angular's commit convention

## Usage

### Development

Download:
```bash
npx degit antonreshetov/vue-vite-ts-boilerplate
# or
pnpx degit antonreshetov/vue-vite-ts-boilerplate
```

Install dependencies:

```bash
yarn 
# or
pnpm i
```

Start dev server:

```bash
yarn dev
# or
pnpm dev
```

## Build

Build for production:

```bash
yarn build
# or
pnpm build
```

Start preview server:

```bash
yarn serve
# or
pnpm serve
```
